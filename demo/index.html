To use ChartMonky, include the core library and any chart modules you want:

<script src="chartmonky.js" defer></script>
<script src="charts/bar.js" defer></script>

Create a container to hold your charts:

<div id="myChart"></div>

Then generate a chart:

const chart = ChartMonky.charts.bar([5, 10, 8], {
    width: 300,
    height: 200
});

document.getElementById("myChart").appendChild(chart);

ChartMonky includes low-level SVG utilities such as: 

ChartMonky.utils.createSVG(width, height)
ChartMonky.utils.createRect(x, y, width, height, color)

Example:

const svg = ChartMonky.utils.createSVG(200, 150);
const rect = ChartMonky.utils.createRect(20, 40, 50, 80, "green");
svg.appendChild(rect);

document.getElementById("utils-demo").appendChild(svg);
