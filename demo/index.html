ChartMonky

A lightweight, modular SVG chart library by Brendon.

üìå Getting Started

Add ChartMonky to your project:

<script src="https://stackmonky.github.io/ChartMonky/js/chartmonky.js"></script>

Create a container:

<div id="myChart"></div>

Render your first chart:


const chart = ChartMonky.charts.bar([5,120,80], {
    width: 300,
    height: 200,
    labels: ["one","two","three"],
    scaleStep: 20,
    colors: ["red","orange","green"]
});

document.getElementById("myChart").appendChild(chart);

üîß SVG Utilities

ChartMonky includes low-level SVG helpers:


ChartMonky.utils.createSVG(width, height)
ChartMonky.utils.createRect(x, y, w, h, color)
ChartMonky.utils.createText(x, y, text, options)
ChartMonky.utils.createLine(x1, y1, x2, y2, options)

Example:


const svg = ChartMonky.utils.createSVG(200, 150);
const rect = ChartMonky.utils.createRect(20, 40, 50, 80, "green");
svg.appendChild(rect);

document.getElementById("utils-demo").appendChild(svg);

‚è±Ô∏è Live Updating Bar Chart

This chart updates every second with random data:


// Auto-update every 1s
function updateLiveBar() {
    const data = Array.from({ length: 5 }, () =>
        Math.floor(Math.random() * 100)
    );

    liveBar.innerHTML = "";

    const chart = ChartMonky.charts.bar(data, {
        width: 350,
        height: 200,
        labels: ["A","B","C","D","E"],
        scaleStep: 10
    });

    liveBar.appendChild(chart);
}

setInterval(updateLiveBar, 1000);
updateLiveBar();

üè≠ Warehouse Map Chart Example

A warehouse map with points:


const warehouseChart = ChartMonky.charts.warehouse({
    width: 400,
    height: 300,
    image: "https://stackmonky.github.io/ChartMonky/images/warehouse.png",
    points: [
        { id: "a1", x: 50, y: 80, color: "red", flash: true }
    ]
});

document.getElementById("warehouse-demo").appendChild(warehouseChart.svg);

// Example: dynamically add
warehouseChart.addPoint({
    id: "a2",
    x: 120,
    y: 150,
    color: "blue",
    flash: false
});

// Example: remove after 2 seconds
setTimeout(() => {
    warehouseChart.removePoint("a1");
}, 2000);
